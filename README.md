## 1. Rice Variety Classification

We have classified 5 Different Varieties of Rice using CNN
The 5 different varieties of Rice are:
1. Arborio
2. Basmati
3. Ipsala
4. Jasmine
5. Karacadag

#### Dataset Link: https://www.muratkoklu.com/datasets/
